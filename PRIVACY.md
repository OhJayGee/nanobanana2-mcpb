# Privacy Policy — Nanobanana Image Studio

**Last updated:** 2026-03-22

## What data is sent externally

This extension sends the following data to Google's Gemini API (`generativelanguage.googleapis.com`):

- **Text prompts** you provide for image generation, editing, or description
- **Image data** (base64-encoded) when using edit, describe, or extract DNA tools
- **Configuration parameters** such as aspect ratio, resolution, and thinking level

## What data stays local

- **Your Gemini API key** is stored locally by Claude Desktop (in the OS keychain) and sent directly to Google — it never passes through Anthropic or any third party
- **Generated images** are saved only to your configured local output directory
- **Visual DNA templates** are bundled locally and never transmitted

## Third-party service

All image generation is performed by **Google's Gemini API**. Your use of this extension is subject to:
- [Google's Terms of Service](https://policies.google.com/terms)
- [Google's Privacy Policy](https://policies.google.com/privacy)
- [Google AI Studio Terms](https://ai.google.dev/terms)

## Data retention

This extension does not store, log, or retain any data beyond the generated output files on your local machine. Google may retain data per their own policies linked above.
